# 🌉 기능 목록

- [x] 게임 시작 문구 출력
- [x] 다리 길이 입력
- [ ] 다리 생성
- [x] 이동할 칸을 입력(U : 위, D : 아래)
- [ ] 입력값과 다리 비교
- [ ] 게임 진행 상황 출력

- [x] 실패시 재시도 여부 (R : 재시도, Q : 종료)

  - [ ] R : 이동할 칸을 입력(처음부터)
  - [ ] Q : 게임 종료 문구 출력

- [ ] 게임 종료 문구 출력
  - [ ] 게임 결과
  - [ ] 성공 여부
  - [ ] 시도 횟수

# ‼️ 예외 처리

- [ ] 다리 길이 3이상 20이하 외의 숫자 입력
- [ ] U,D외의 문자 입력
- [ ] 재시작/종료 여부 R/Q 외의 값 입력
